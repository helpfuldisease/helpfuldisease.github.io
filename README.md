# helpfuldisease.github.io

这是我的个人主页项目，用于展示我的作品、能力、业务范围和联系方式。

## 网站目标

建立一个面向 freelancer job 的个人网页，让潜在客户可以快速理解：

- 我是谁以及我的风格
- 我能提供什么服务
- 我做过哪些作品
- 我的合作方式
- 如何联系我

## 当前技术路线

当前阶段使用 GitHub Pages 托管静态网页。

主要网页文件：

- `index.html`：网站首页
- `assets/`：图片、作品截图、参考素材
- `docs/`：项目定位、信息架构、视觉方向和开发记录

后续如果需要长期写博客，可以再评估是否引入 Hexo、Astro 或其他静态网站框架。

## 设计参考

主要结构参考：

- Gabrielle Adam: https://gabrielleadam.com/

核心借鉴点：

- 左侧固定自我介绍
- 右侧滚动展示作品
- 信息路径清楚
- 让访问者快速找到服务、作品和联系方式
