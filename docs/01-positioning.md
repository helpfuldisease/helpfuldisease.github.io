# 01 - Positioning

## 项目定位

这是一个面向 freelancer job 的个人主页，不是传统简历页，也不是纯博客。

它的核心目标是帮助潜在客户快速判断：

- 我是否理解他们的问题
- 我是否具备完成项目的能力
- 我的审美和工作方式是否适合合作
- 如何联系我开始合作

## 目标受众

主要访问者：

- 潜在客户
- 独立创业者
- 早期团队
- 个人品牌主理人
- 需要网页、作品集、AI 工具或自动化流程的人

## 我希望传达的印象

关键词：

- 清晰
- 可靠
- 高级
- 自然
- 有叙事感
- 有执行力
- 能把模糊想法落地

## 服务方向草案

- 个人主页与作品集设计
- Landing page 设计与实现
- AI 工具与自动化工作流
- 产品原型
- 内容结构与品牌叙事
- 网页视觉设计与前端实现

## 后续需要补充

- 我的真实姓名或展示名
- 一句话身份介绍
- 3-5 个代表作品
- 联系方式
- GitHub / LinkedIn / 邮箱 / 预约链接
