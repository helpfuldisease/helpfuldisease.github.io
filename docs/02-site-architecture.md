# 02 - Site Architecture

## 首页结构

网站采用双栏结构：

- 左侧：固定自我介绍和联系方式
- 右侧：滚动作品和服务内容

参考 Gabrielle Adam 的个人主页结构，但视觉和内容会根据 freelancer 接单目标重新设计。

## 左侧固定区域

左侧需要包含：

1. 姓名 / 展示名
2. 一句话身份介绍
3. 简短自我介绍
4. 服务范围
5. 合作状态
6. 联系方式
7. 社交链接

## 右侧滚动区域

右侧内容顺序建议：

1. Featured Works
2. Services
3. Process
4. About / Working Style
5. Contact

## 作品卡片内容

每个作品建议包含：

- 项目名称
- 项目类型
- 一句话简介
- 我负责的内容
- 结果或亮点
- 截图或链接

## 手机端结构

手机端不使用左右双栏。

顺序改为：

1. 自我介绍
2. 联系按钮
3. 代表作品
4. 服务范围
5. 合作流程
6. 联系方式
